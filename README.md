# Recipe-app

open a new terminal:

--> git clone https://github.com/NITHIN3387/Recipe-app.git

--> cd Recipe-app

--> npm i 

--> npm run build

This will run the server at the port you mentioned in .env file

open a new terminal:

--> cd Recipe-app/Final-Milestone

--> npm i

--> npm start

This will run the frontend part of the website
